# janalytics-react-native

## Install

```
npm install janalytics-react-native --save
npm install jcore-react-native@1.1.5 --save
react-native link
```

## Usage

- Import JShareModule:

  > your component.js

  ```
  ...
  import JAnalyticsModule from 'janalytics-react-native';
  ```

- You can refer to [example](./example)
